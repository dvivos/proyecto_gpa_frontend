El proceso auxiliar de cambio permisos de un usuario elimina los permisos de un usuario y le asigna los de un usuario copia indicado. Este proceso realiza las validaciones necesarias para realizar el cambio correctamente.

Hay un formulario que solicita el usuario que va a heredar los permisos y el usuario del que se van a copiar los permisos.

Los usuarios deben estar activos y tener la misma Unidad Tramitadora.
